How the Analysis Works

This tool analyzes your keyword lists to find SEO opportunities. Here's a summary:

  • Optimise Existing: For content ranking between positions 5-20.
  • Create New Page: For content ranking below position 20 or for new keyword gaps.
  • No Action: For content already ranking in the top 4.
Read more

Keyword Consolidation: First, the tool analyzes your "Full Topic Attack List" and groups semantically similar keywords together using AI-powered embeddings. For each group, it sums up their search volumes and picks the keyword with the highest individual volume as the main representative for that group. This prevents suggesting actions for many minor variations of the same core topic.

Semantic Matching: Next, it compares each consolidated main keyword against your "Existing Pages & Keywords" list to find the best semantic match, even if the wording isn't identical.

Action Logic: Based on how closely the topic matches your existing content and the ranking position of that content, it recommends one of the following actions:

  • Optimise Existing: This is suggested if a strong semantic match is found and your existing page for that topic ranks between positions 5 and 20.
  • Create New Page: This is suggested in two scenarios:
    • If a strong semantic match is found, but the existing page ranks below position 20.
    • If no strong semantic match is found in your existing content, indicating a content gap.
  • No Action: If a strong match is found and the page is already ranking in the top 4 positions, no action is recommended as it's already performing well.

CSV Upload Requirements

For the tool to work correctly, your CSV files must contain specific columns.

Step 1: Existing Pages & Keywords CSV

This file must include the following columns:

  • Keyword
  • Current position
  • Current URL

TIP: Use an SEO tool to get clean data. For example, in Ahrefs, you can find this by going to Site Explorer > Organic Keywords and exporting the list.

Step 2: Full Topic Attack List CSV

This file must include the following columns:

  • Keyword
  • Volume

TIP: Use an SEO tool like Ahrefs' Keyword Explorer to export full topics and get a view of what's out there to potentially target. You can also pre-filter your upload for brand/non-brand, informational, transactional, etc. for cleaner outputs.
